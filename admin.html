資料與覆寫

商品清單來自 data.js 的 GACHA_PRODUCTS，若設定了 GACHA_DATA_URL 則優先使用遠端 JSON。 「儲存設定」寫入上架覆寫（鍵：）。本機新增／覆寫的商品存於（鍵：），同編號時覆寫雲端／本檔條目。 試算表不會自動寫入；需在下方「新增商品」勾選同步寫入試算表並填 ADMIN_TOKEN（與 Apps Script 指令碼屬性相同）。 可將 GACHA_AUTO_SYNC_SHEET 設為 true 以預設勾選同步（仍須填權杖）。

操作方式

以滑動開關切換上下架（綠色＝上架，灰色＝下架）。可用搜尋縮小列表；批次可一鍵全部上架或全部下架（仍須按「儲存設定」才會寫入本機）。 若要改回只跟隨雲端或 data.js 的上下架，請按「清除本機覆寫」。若要刪除本機新增的商品條目，請在列表「本機」欄按「移除」，或按「清除本機新增商品」。

新增商品（本機）

必填：名稱、系列。編號留空則自動產生。規格為 JSON 陣列，可留空。與試算表同編號時會以本表單覆寫該筆（仍須「儲存設定」才會寫入上下架覆寫）。 若在 data.js 設定 Cloudinary（cloud name、unsigned preset、選填 folder），選圖會自動上傳雲端並填入主圖／圖集網址；未設定則改為本機壓縮（Data URL）。
